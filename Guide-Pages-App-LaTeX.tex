\begin{itemize}

  \item Mot de passe (min 8 caractères, 1 majuscule, 1 chiffre, 1 spécial)

  \item Checkbox: ``J'accepte les CGU et Politique de confidentialité''
  \item Bouton: ``Continuer''
\end{itemize}

\subsubsection{Étape 2: Profil Freelancer}
\begin{itemize}
  \item Prénom
  \item Nom
  \item Photo de profil (upload)
  \item Bio courte (max 200 caractères)
  \item Titre professionnel (ex: ``Développeur React Senior'')
  \item Pays + Ville
  \item Bouton: ``Continuer''
\end{itemize}

\subsubsection{Étape 3: Compétences \& Tarifs}
\begin{itemize}
  \item Ajouter compétences (min 1, max 10)
  \begin{itemize}
    \item Sélectionner de liste prédéfinie ou ajouter custom
    \item Niveau (Débutant / Intermédiaire / Avancé / Expert)
  \end{itemize}
  \item Tarif horaire minimum (montant en MAD)
  \item Disponibilité (heures/semaine)
  \item Bouton: ``Finaliser''
\end{itemize}

\subsection{Éléments Additionnels}

\subsubsection{Side Panel (Desktop)}
Progression: étape 1/3, 2/3, 3/3 avec barres de progression.

Ce document détaille \textbf{42 pages/écrans principaux} d'une plateforme complète de freelancing comme AtlasFreelanceApp. Chaque page inclut:

\begin{itemize}
  \item \textbf{URL de route}

  \item \textbf{Éléments visuels}: layouts, sections, widgets

\end{itemize}
